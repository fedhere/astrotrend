This is a natural language analysis of the Arxiv.org postings from the past week.

Wait a few seconds while statistics are being computed....

The color of the wedges represents the field the word most commonly appears in. Each word is linked to each word it appears paired more than once in the week's Arxiv.org titles.

Roll over each wedge to see more information about each word included in the graph, and on the arc to for some information on the word co-occurrence.

The code is on public on github and it is very much a work in progress. Text is parsed in gathered and python and the graphics is created in d3, which is awesome, and it leverages heavily off this awesomegreat blogpost by Steve Hall.
